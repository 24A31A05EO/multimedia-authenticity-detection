\# Frontend Overview  

Multimedia Authenticity Detection System



\## Technology Stack

\- React (Vite)

\- TypeScript

\- Tailwind CSS

\- shadcn/ui

\- Axios



---



\## Application Pages

1\. Dashboard  

2\. Email Detection  

3\. URL Detection  

4\. Image Detection  

5\. Audio Detection  

6\. Video Detection  



---



\## Dashboard Features

\- Total scans count

\- Fake detection count

\- Accuracy rate

\- Live updates after each detection



---



\## Detection Flow

1\. User selects detection type

2\. Uploads content or enters text/URL

3\. Frontend sends request to backend API

4\. Backend returns result

5\. Dashboard stats update automatically
